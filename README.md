导师在教授Python课程时，发现同学们的作业基本全对，但是考试成绩下70分的特别多，于是想做一个辅助Python课程教学的系统软件，它要满足三个需求：1.遏制学生抄袭成风的现象2.收集学生在编写代码过程中的报错信息3.通过报错信息定位到相应知识点，基于知识点构建个人和全班的知识画像、分析知识弱点。
